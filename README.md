# Canadian-Cannabis-Survey-Youth-Trends-2023-2024
A Python project to analyze year-over-year changes in cannabis use, knowledges, behaviours among Canadians between 16 to 25 years old using data from the 2023 and 2024 Canadian Cannabis Surveys.  

This project aims to identify shifts in national patterns with this age group. Hence, inform policy, public health and business insights and improve Canadian teenagers' and young adults' Cannabis abuse.
